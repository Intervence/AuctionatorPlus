# AuctionatorPlus Addon for Wrath of the Lich King 3.3.5

A backported version of the 3.1.5 release of the Auctionator addon, adapted for World of Warcraft: Wrath of the Lich King patch 3.3.5.

## Key Features

*   View and manage auctions
*   Quickly post items to the Auction House with configurable pricing strategies
*   Easily search for items and compare prices
*   Automatically undercut the lowest current auction
*   Bulk posting and cancelling of auctions
*   Detailed pricing history
